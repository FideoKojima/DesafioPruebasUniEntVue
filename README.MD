## 🏗️ Estructura del Proyecto

```
📦 vue-unit-testing
├── 📁 src
│   ├── 📁 views
│   │   ├── 📄 HomeView.vue
│   │   ├── 📄 AboutView.vue
│   │   └── 📄 ContactView.vue
│   └── 📁 router
│       └── 📄 index.js
└── 📁 tests
    └── 📁 unit
        └── 📁 views
            ├── 📄 HomeView.spec.js
            ├── 📄 AboutView.spec.js
            └── 📄 ContactView.spec.js
```

## 🚦 Comandos Principales

```bash
# Instalar dependencias
$ npm install

# Ejecutar servidor de desarrollo
$ npm run serve

# Ejecutar pruebas unitarias
$ npm run test:unit

# Construir para producción
$ npm run build
```
